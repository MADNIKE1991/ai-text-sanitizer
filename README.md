# AI Text Sanitizer (Chrome Extension & Local App)

Micro-утилита для мгновенной нормализации типографики в текстах, сгенерированных нейросетями. 

## ⚠️ Проблема
Языковые модели часто выдают текст с избыточно академическим форматированием: используют кавычки-елочки (« ») и длинные тире (—), которые могут ломать верстку при прямом переносе в веб-интерфейсы, админки или базовые текстовые редакторы.

## 💡 Решение
Автономный легковесный инструмент (доступен как локальное приложение и расширение для Chrome), который в один клик проводит Regex-очистку буфера обмена. 

Скрипт выполняет две точечные операции:
- Трансформирует кавычки-елочки (« ») в стандартные прямые кавычки (" ").
- Заменяет длинные тире (—) на обычные дефисы (-).

## 🛠 Стек
- JavaScript (ES6+)
- Chrome Extensions API V3
- HTML5 / CSS3

## 🚀 Установка (Chrome)
1. Скачайте репозиторий.
2. Откройте `chrome://extensions/` в браузере.
3. Включите "Developer mode" (Режим разработчика) в правом верхнем углу.
4. Нажмите "Load unpacked" и выберите папку расширения.
