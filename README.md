# AI Text Sanitizer (Chrome Extension & Local App)

> Micro-утилита для мгновенной нормализации типографики в текстах, сгенерированных нейросетями, и безошибочного переноса их в кодовую базу или веб-интерфейсы.

---

## 🚀 Архитектура и стек

* **Платформа:** Chrome Extensions API V3 & Standalone Web App.
* **Core:** Vanilla JavaScript (ES6+) — Regex-обработчик буфера обмена (Clipboard API).
* **UI:** HTML5, CSS3 (адаптивный минималистичный интерфейс).

---

## 🛠 Проблема и Решение

* **Проблема:** Языковые модели часто выдают текст с академическим форматированием: кавычками-елочками (`« »`) и длинными тире (`—`). При переносе такого текста в код, переменные, базу данных или админки верстка ломается, а скрипты падают с синтаксическими ошибками.
* **Решение:** Автономный легковесный инструмент (доступен как расширение для Chrome и локальное веб-приложение), который в один клик выполняет Regex-очистку текста в буфере обмена:
  1. Трансформирует кавычки-елочки (`« »`) в стандартные прямые кавычки (`" "`).
  2. Заменяет длинные тире (`—`) на обычные дефисы (`-`).

## 📦 Установка и запуск (Chrome Extension)

1. Клонируй репозиторий:
   ```bash
   git clone [https://github.com/MADNIKE1991/ai-text-sanitizer.git](https://github.com/MADNIKE1991/ai-text-sanitizer.git)
   cd ai-text-sanitizer
   ```

2. Установи необходимые зависимости (если используются билдеры/инструменты сборки):
   ```bash
   npm install
   ```

3. Открой в браузере Chrome: `chrome://extensions/`
4. Включи **«Режим разработчика»** (Developer mode) в правом верхнем углу.
5. Нажми **«Загрузить распакованное расширение»** (Load unpacked) и выбери папку проекта.
