# AI Text Sanitizer (Chrome Extension & Local App)

Micro-утилита для мгновенной очистки текстов, сгенерированных нейросетями. 

## ⚠️ Проблема
Языковые модели (ChatGPT, Claude, Gemini) часто выдают текст с избыточным форматированием: машинные кавычки, маркеры списков, елочки (« ») и лишние тире, которые ломают верстку при переносе в веб-интерфейсы или редакторы.

## 💡 Решение
Автономный инструмент (доступен как локальное приложение и расширение для Chrome), который в один клик проводит глубокую Regex-очистку буфера обмена:
- Трансформирует «елочки» и специфические символы в стандартные кавычки.
- Нормализует тире и дефисы.
- Удаляет мусорный синтаксис LLM-моделей.

## 🛠 Стек
- JavaScript (ES6+)
- Chrome Extensions API V3
- HTML5 / CSS3

## 🚀 Установка (Chrome)
1. Скачайте репозиторий.
2. Откройте `chrome://extensions/` в браузере.
3. Включите "Developer mode" (Режим разработчика) в правом верхнем углу.
4. Нажмите "Load unpacked" и выберите папку расширения.
